# java-compound-interest-calculator
A simple Java console application that calculates compound interest based on the principal amount, interest rate, compounding frequency, and investment period. Built to practice Java fundamentals, user input, and mathematical calculations.


# Compound Interest Calculator
A simple Java console application that calculates compound interest based on:

- Principal amount
- Interest rate
- Compounds per year
- Number of years

## Technologies

- Java
- Scanner
- Math.pow()

## What I learned

- User input with Scanner
- Variables
- Mathematical calculations
- Using Math.pow()
